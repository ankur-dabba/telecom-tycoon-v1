# 📡 Telecom Tycoon

A Paperclips-inspired idle game where you build a telecom company from nothing — generate data, sign up customers, hire suppliers, and grow to 10,000 customers.

## 🎮 Play it

👉 **[Click here to play](https://ankur-dabba.github.io/telecom-tycoon-v1/)**

## 🕹 How to play

1. Click **Generate Data** to start earning money
2. Once you have enough GB, buy some **Routers** from the Hardware section
3. Use your routers to **Partner with an LCO** (a local internet provider) — this gets you customers
4. Hire a **Bandwidth Supplier** so data generates automatically without clicking
5. Keep growing until you hit **10,000 customers**, then Prestige and start again stronger

## 🛠 Want to modify or add to the game?

The entire game is written in a single file: `game/index.html`. There's no complicated setup — just open that file in your browser and it works.

To make changes:

1. Click **Fork** at the top of this page (makes your own copy)
2. Download or edit `game/index.html`
3. Open it in your browser to test your changes
4. When you're happy, send a **Pull Request** and we'll review it

**Some ideas if you want to contribute:**
- Save and load your game progress
- Make it work better on phones
- Add sound effects
- Add new types of events or disasters
- Add a high score / leaderboard
