# 📡 Telecom Tycoon

A [Paperclips](https://www.decisionproblem.com/paperclips/)-inspired idle game about building a telecom company from nothing.

## 🎮 Play it

👉 **[Play online](https://YOUR_USERNAME.github.io/telecom-tycoon/)** *(link live after a minute)*

Or download `game/index.html` and open it in any browser — no install needed.

## 🕹 How to play

1. **Generate Data** — click to mine GB and build starting funds
2. **Buy Routers** — from the Hardware Manufacturing section (unlocks at 50 GB)
3. **Partner with LCOs** — local cable operators bring you customers (unlocks at 3 routers)
4. **Hire Bandwidth Suppliers** — auto-generate data so you don't have to click
5. **Grow to 10,000 customers** — then Prestige and do it all again, faster
